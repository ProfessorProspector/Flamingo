Flamingo
========

![The amazing Pink Flamingo!](docs/flamingo.png)

Flamingo Minecraft mod for Minecraft 1.7.10, 1.8.8 and 1.9. Adds a Pink Flamingo.

###Downloads

Downloads can be found [here](https://github.com/Vexatos/Flamingo/releases).
